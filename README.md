# devops-monitoring-project
## About
This project is designed to simulate a **real-world DevOps workflow** for a Python application. It focuses on building a fully containerized environment, implementing monitoring and observability, and setting up automated CI/CD pipelines.

This project was **entirely developed and implemented by me** to showcase practical skills in:  
- Deploying and managing containerized applications using Docker and Docker Compose  
- Monitoring application metrics in real-time with Prometheus and visualizing them in Grafana  
- Automating builds, tests, and deployments with Jenkins pipelines  
- Following professional practices for configuration, networking, and persistent storage
